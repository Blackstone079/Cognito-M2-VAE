# Experiments

This folder keeps the main experiment snapshots from the M2-VAE development process.

The reported paper model is in `../Paper Model/`. These folders are here mainly to show the major directions that were tried along the way.

In broad terms:
- `v8` is the repaired three-head baseline on the original feature line
- `v9` and `v10` are training/optimisation follow-ups
- `v12` is the count/chirality/tox24 representation branch
- `v14` to `v17` are later follow-up branches on the stronger feature line
- `v11` is kept as an intermediate code snapshot for traceability

This is not meant to be a polished experiment registry. It is just the simplest way to keep the important code history visible.
