# Cognito M2-VAE

This repository is a cleaned code snapshot for the Cognito VAE work.

The main paper configuration lives in `Paper Model/`. The `Experiments/` folder keeps a few of the larger model-development steps so it is still possible to see how the work evolved.

Only code is included here. Large outputs and project artefacts are intentionally left out, including:
- result folders
- checkpoints
- datasets and memmaps
- logs
- generated report figures and tables

So the repository is meant to show the codebase and the main experiment trail, not to act as a full run archive.
